Hướng dẫn

Lưu ý: Hướng dẫn dưới đây chỉ trình bày tóm tắt cách khởi động hệ thống.

Tóm tắt hệ thống
Hệ thống sử dụng Raspberry Pi 4 kết hợp với camera(pi v2.1) làm trung tâm điều khiển kết nối với Arduino Uno. Website này được xây dựng để có thể sử dụng làm bảng điều khiển, giám sát hệ thống trên máy tính hoặc thiết bị di động.
Từ trang Bảng Điều Khiển Raspberry Pi có thể nhận lệnh điều khiển từ người dùng để khởi động/dừng hệ thống và gửi yêu cầu(request HTTP) tới Raspberry Pi. Sau khi nhận được lệnh điều khiển, Raspberry Pi giao tiếp với Arduino Uno thông qua serial. Raspberry Pi là một máy tính nhúng, nó sẽ truyền hình ảnh từ camera tới trình duyệt web. Nó cũng chạy một mạng phân loại CNN (sử dụng tensorflow lite) để phân loại. Mạng phân loại chỉ được chạy khi Raspberry Pi nhận được lệnh từ Arduino Uno (khi hộp được phát hiện bởi cảm biến IR đầu tiên).

Khởi động hệ thống
Kết nối với Raspberry Pi 4 qua cổng display HDMI hoặc qua SSH. Đề tài này kết nối với Raspberry qua SSH sử dụng VNC (chia sẻ giao diện màn hình) để kết nối Rasp Pi với Laptop.
Sử dụng cổng Ethernet trên Raspberry PI để kết nối với Laptop, sau đó Ping kiểm tra IP của Raspberry Pi từ Command Prompt (Windows): 

                           ping raspberrypi.mshome.net

Lưu ý: Cần phải bật VNC trong Raspberry Pi trước.

Sau đó:
Trên terminal Raspberry Pi truy cập đến thư mục chứa Project. Trên giao diện dòng lệnh: 
                          
                           python3 app.py -p /dev/ttyACM0 -s 0
Trong đó:
  • "-p /dev/ttyACM0" là device_name của Arduino Uno khi kết nối tới Rasp
  • -s 0 thực hiện reset folder (chứa ảnh được camera chụp của vật thể để phân loại)

    • Sau khi khởi động hệ thống, Raspberry sẽ truyền hình ảnh và kết quả phân loại tới website thông qua Bảng Điều Khiển
    Trên cổng 5000(http://Pi_IP_Address:5000)
    Mục đích của khoá luận này là xây dựng một hệ thống có thể phân loại hoa quả. Do thời gian hạn chế trong thời gian thực hiện, khoá luận trước hết xây dựng mô hình xử lý ảnh kết hợp với máy tính nhúng để phân loại. Bên cạnh đó hình ảnh của đối tượng cần xử lý sẽ được giám sát trực tiếp trên màn hình hiển thị cùng số lượng sản phẩm đã qua.
